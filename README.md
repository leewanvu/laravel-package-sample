# Laravel Package Sample (Laravel 5.6 to Laravel 5.8)

## Requirements

* PHP 7.1.3+
* Laravel 5.6+

## Installation

1) Install the package by running this command in your terminal/cmd:
```
composer require leewanvu/laravel-package-sample
```
